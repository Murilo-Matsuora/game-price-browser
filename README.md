# NEXARDA Game Price Browser

A web-based video game price comparison tool built with **Python** and **Flask** that integrates with the **NEXARDA API**. Users can search for titles and view current pricing.

## Features

* **Real-Time Search**: Search the NEXARDA database for any video game using a simple search interface.
* **Price Comparison**: View the lowest and highest prices for specific titles.
* **Dynamic Results**: Browse search results that include game titles, cover images, and current lowest prices.
* **Responsive UI**: Built with **Bootstrap 5** to ensure the site works on mobile and desktop devices.

## Project Structure

* **Backend**: Python with Flask.
* **API Integration**: Requests library for communicating with the NEXARDA API.
* **Frontend**: HTML, Jinja2, and Bootstrap 5.
